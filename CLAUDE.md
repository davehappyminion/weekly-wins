A beautiful dashboard to track and celebrate your weekly accomplishments!

## Features
- **Log Wins** - Add accomplishments in 4 categories: Work, Health, Personal, Learning
- **GitHub Activity** - Auto-fetches recent GitHub events for @nikilster
- **Streak Tracking** - See how many consecutive days you've logged wins
- **Win Distribution** - Visual breakdown of wins by category
- **Motivational Quotes** - Random inspiring messages

## Tech Stack
- Next.js 16
- TypeScript
- Tailwind CSS
- Vercel

## Deployment
- **URL:** https://wins.davehappyminion.com
- **Vercel Project:** weekly-wins
- **GitHub:** https://github.com/davehappyminion/weekly-wins

## How It Works
1. Wins are stored in browser localStorage
2. GitHub activity is fetched from the public API
3. Streak is calculated based on consecutive days with logged wins
